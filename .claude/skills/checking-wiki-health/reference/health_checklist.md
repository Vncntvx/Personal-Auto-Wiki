## 检查流程

### 1. 统计信息

- [ ] 统计 Concepts 数量
- [ ] 统计 Entities 数量
- [ ] 统计 Sources 数量
- [ ] 统计 Outputs 数量
- [ ] 计算平均质量分数

### 2. 孤立页面检测

- [ ] 扫描所有页面
- [ ] 提取所有 wiki-links
- [ ] 识别未被引用的页面
- [ ] 评估孤立页面价值

### 3. 未索引页面检测

- [ ] 列出 Concepts 目录文件
- [ ] 列出 Entities 目录文件
- [ ] 列出 Sources 目录文件
- [ ] 列出 Outputs 目录文件
- [ ] 与 Index.md 对比

### 4. 概念缺口检测

- [ ] 扫描所有 `[[Concepts/xxx]]` 链接
- [ ] 扫描所有 `[[Entities/xxx]]` 链接
- [ ] 检查目标文件是否存在

### 5. 重复概念检测

- [ ] 使用 qmd 搜索语义相似的概念页面
- [ ] 检查 `canonical: false` 的别名页面是否正确指向规范条目
- [ ] 识别未建立别名关系但高度相似的概念对
- [ ] 评估是否需要合并或建立别名关系

### 6. 矛盾标注检测

- [ ] 搜索 `> [!warning] 矛盾标注`
- [ ] 收集所有矛盾位置
- [ ] 评估矛盾严重性
- [ ] 判断是否需要重写相关 Concepts/Entities 的"当前综合"部分

### 7. 未同步文件检测

- [ ] 调用 detecting-resources-sync
- [ ] 识别新增文件
- [ ] 识别变更文件
- [ ] 识别删除文件
- [ ] 识别重命名文件

### 8. 认知质量检测（量化）

- [ ] 读取/计算页面 stats 字段
- [ ] 计算质量分数 (quality_score)
- [ ] 识别低质量页面 (score < 0.5)
- [ ] 识别来源变更（modified）但未重审的页面
- [ ] 识别高价值待晋升的 Sources
- [ ] 识别重复或相似的 Outputs

### 9. Schema 时效检测

- [ ] 读取 VERSION 文件与 CHANGELOG.md 最新条目
- [ ] 检查 AGENTS.md、Skills、Reference 文件的修改时间是否晚于 CHANGELOG 最新记录
- [ ] 识别未记录在 CHANGELOG 中的 Schema 变更
- [ ] 检查 VERSION 是否反映当前变更级别

## 量化指标

| 指标 | 计算公式 | 阈值 | 判定 |
|------|----------|------|------|
| 来源密度 | Sources 引用数 | ≥ 3 | 高密度 |
| 综合深度 | 字数 ÷ Sources 数 | ≥ 200 | 深度充足 |
| 引用深度比 | 入链数 ÷ 字数 × 1000 | ≥ 0.5 | 被重视 |
| 定义完整度 | 有「定义」章节且 ≥ 50 字 | 是 | 完整 |
| 时效性 | 当前日期 - updated | ≤ 180 天 | 时效良好 |

## 质量评分公式

```
quality_score = 0.20×来源密度分 + 0.30×综合深度分 + 0.15×引用深度比分 + 0.20×定义完整分 + 0.15×时效分
```

**评分标准**：
- 来源密度分：≥5 源得 1.0，≥3 源得 0.7，≥1 源得 0.4，无源得 0.0
- 综合深度分：≥300 字/源得 1.0，≥200 得 0.8，≥100 得 0.5，<100 得 0.3
- 引用深度比分：≥0.8 得 1.0，≥0.5 得 0.8，≥0.2 得 0.5，<0.2 得 0.3
- 定义完整分：有定义章节且 ≥50 字得 1.0，否则 0.5
- 时效分：≤30 天得 1.0，≤90 天得 0.8，≤180 天得 0.6，>180 天得 0.3

**质量判定**：
- `quality_score ≥ 0.8`：高质量，可作为稳定知识页
- `0.5 ≤ quality_score < 0.8`：中等质量，需持续完善
- `quality_score < 0.5`：低质量，应优先改进或标记待重审

## 问题优先级

| 优先级 | 问题类型 | 处理建议 |
|--------|----------|----------|
| P0 | 未同步文件 | 立即摄取 |
| P0 | 低质量页面 | 补充来源、扩展定义 |
| P1 | 概念缺口 | 创建缺失页面 |
| P1 | 重复概念 | 合并或建立别名关系 |
| P1 | 矛盾标注 | 人工审核解决 |
| P1 | 来源变更（modified）待重审 | 检查并更新相关综合页 |
| P2 | 未索引页面 | 更新 Index.md |
| P3 | 孤立页面 | 评估后处理 |
| P3 | Schema 时效 | 更新 CHANGELOG.md 和 VERSION |

## 定期检查建议

- **每日**：检查未同步文件（如有摄取活动）
- **每周**：完整健康检查（含认知质量）
- **每月**：清理孤立页面、更新索引、重审低质量页面
